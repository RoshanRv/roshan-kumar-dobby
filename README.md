# Picseum

A webapp for pic lovers to upload their pics.

```
Server is hosted on Render (Free Tier), Thus It will take Server some time to start.
```

## Technology Used

### Frontend

-   React JS
-   Typescript
-   TailwindCSS
-   Zustand

### Backend

-   Node JS
-   Express JS
-   Mongo DB
-   Mongoose
-   Bcrypt
